Voylera - Emotions-Mapping

Projektbeschreibung

Voylera ist eine einzigartige Web-Anwendung, die es Nutzern ermöglicht, ihre Gefühle und Gedanken anonym auf einer interaktiven Weltkarte zu teilen. Das Projekt zielt darauf ab, eine globale emotionale Momentaufnahme zu schaffen und Menschen durch gemeinsame Gefühle zu verbinden.

Funktionen

  • Interaktive Weltkarte zur Visualisierung von Emotionen
  • Anonyme Beitragserstellung mit Emotions-Tags
  • Filter- und Suchfunktionen nach Ort und Gefühl
  • Echtzeit-Aktualisierungen der Gefühlskarte
  • Responsives Design für eine optimale Darstellung auf allen Geräten

Verwendete Technologien

PHP MySQL JavaScript (ES6+) Tailwind CSS Leaflet.js HTML5

Europa.tips - Reiseblog

Projektbeschreibung

Europa.tips ist mein persönlicher, mehrsprachiger Reiseblog, der auf einer Headless-WordPress-Architektur basiert. Anstatt ein Standard-Theme zu verwenden, werden die Inhalte über eine benutzerdefinierte PHP/PDO-Schnittstelle direkt aus der MySQL-Datenbank abgerufen. Dies gewährleistet maximale Performance und volle gestalterische Freiheit.

Funktionen

  • Vollständige Unterstützung für 9 Sprachen
  • Maßgeschneidertes Frontend für einzigartiges Design und hohe Geschwindigkeit
  • Headless-Ansatz mit WordPress als Backend-CMS
  • Direkte und performante Datenabfrage via PHP/PDO
  • Starke SEO-Optimierung für internationale Sichtbarkeit
  • Vollständig responsives Design für alle Endgeräte

Verwendete Technologien

Headless WordPress PHP PDO (PHP Data Objects) MySQL JavaScript SEO

Sitefex - Bio-Link Plattform

Projektbeschreibung

Sitefex ist eine mit dem Laravel-Framework entwickelte Plattform, die es Nutzern, insbesondere Influencern und Kreativen, ermöglicht, schnell und einfach eine personalisierte Bio-Link-Seite zu erstellen. Die Anwendung bietet eine intuitive Benutzeroberfläche zur Verwaltung von Links, sozialen Medien und persönlichem Branding.

Funktionen

  • Einfacher Editor zur Erstellung von Bio-Seiten
  • Umfassende Anpassungsmöglichkeiten für Farben, Schriftarten und Layout
  • Unbegrenzte Anzahl an Links zu Webseiten und sozialen Profilen
  • Sicheres Benutzer- und Authentifizierungssystem
  • Robuste und skalierbare Backend-Architektur mit Laravel

Verwendete Technologien

Laravel PHP MySQL Blade Templates JavaScript Tailwind CSS
